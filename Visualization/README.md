# Visualization

This folder contains R scripts for visualizing RNA-seq and whole-genome sequencing (WGS) data from the study of chromosomal deletions and Fusarium wilt resistance in banana somaclonal variants.

## Contents

| Script                | Figure(s) | Description                                                                 |
|-----------------------|-----------|-----------------------------------------------------------------------------|
| PNAS_FIG2DF_S4.R      | Fig. 2D, 2F, S4 | TC5 DEG barplot, GO enrichment, heatmap of TPMs in deletion region             |
| PNAS_Fig3D.R          | Fig. 3D    | Chromosome 5 allele ratio visualization from WGS VCFs                       |
| PNAS_Fig3F.R          | Fig. 3F    | Heatmap of variant allele ratios in FM deletion region                      |
| PNAS_Fig4D.R          | Fig. 4D    | Chromosome 5 allele ratio visualization across cultivars                    |
| PNAS_Fig4E&S15.R      | Fig. 4E, S15 | DEG summary and chromosome-level plots                                      |
| PNAS_FigS5.R          | Fig. S5    | Phylogenetic tree and domain structures of ACA proteins                     |
| PNAS_FigS7.R          | Fig. S7    | Whole-genome relative coverage plots mapped to DH Pahang                       |
| PNAS_FigS10.R         | Fig. S10   | Genome-wide relative coverage plots mapped to Cavendish (Li et al., 2023)      |
| PNAS_FigS11.R         | Fig. S11   | Genome-wide relative coverage plots mapped to Cavendish (Huang et al., 2023)   |
| PNAS_FigS15A.R        | Fig. S15A  | Chromosome 5 synteny visualization (genome alignment)                       |
| PNAS_FigS15B.R        | Fig. S15B  | Gene bin count within FM deletion region                                    |
| PNAS_FigS15C.R        | Fig. S15C  | Collinearity visualization (MCScanX results)                                |

## Data Sources

- Input files (VCF, BigWig, GFF, etc.) are generated by upstream analysis pipelines in `RNA_process/` and `WGS_process/`.
- See each script for specific file paths and requirements.

## How to Use

1. Open the desired script in R (RStudio or VS Code).
2. Ensure required input files are available and paths are correct.
3. Install required R packages (see below).
4. Run the script to generate plots and figures.
